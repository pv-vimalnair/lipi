How do I add a custom theme to Lipi?
Three steps: drop your illustration into src/shared/assets/themes/, add an entry to the THEMES array in tokens.css, and pick it from Settings → Theme. The active tab picks up the new background automatically via the --active-img CSS variable — no other component changes needed.
